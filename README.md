# Gym Managements System.


- End to end management system for gym organizations.

## To Run The Project

### Clone the repo 
- Clone this repo and move to project directory

#### Install Node Modules 

```
npm install 

```

#### Run On Local 
```
npm run dev
```

#### Run on Production
```
npm run start
```

#### Create a  Build
```
npm run build
```
